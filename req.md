สร้าง website agentSkill marketplace ซึ่งเป็น platform สำหรับให้คน upload skill ขึ้นและ user คนอื่นๆมองเห็น เข้าไปดู detail ภายในหรือกดดาวโหลดได้

มี user ทั้งหมด 2 role คือ 1.user 2.admin

ระบบ group ตัว group จะมีความสัมพันเป็น many to many กับ user โดย default user จะไม่มี group อะไรเลยและ หาก user อยู่ใน group ไหน user สามารถมองเห็น skill ของ group นั้นๆได้แม้ skill นั้นจะถูก set เป็น privcate ก็ตาม แต่ถ้าเป็นคนที่อัพ skill นั้นหรือ admin จะเห็น skill นั้นๆได้เลย

ในหน้า frontend มีหน้าทั้งหมด 5 หน้าคือ

1. register/login โดยใช้เป็น register และ login แบบ local เอาข้อมูลใส่ database และเทียบ database ตรงๆ แต่มีการ encrypt ข้อมูลนิดนึงโดยใช้ database เป็น filebase หรือ serverless นะ
2. หน้า marketplace ให้เป็นหน้าที่ลิส skill.md ทั้งหมดโดยจะเห็นเป็นรูป skill พร้อม tag และ category ของ skill นั้นๆและสามารถกดที่ tag หรือ category เพื่อ filter  skill ที่ list ออกมาได้โดยมันจะ list ออกมาแค่ที่ tag ตรงหรือ category ตรงกับที่เราคลิกเท่านั้น(คลิก tag หรือ category ซ้ำเพื่อเลิก filter) และสกิลจะมีกรอบอยู่ 2 สี สีแดงสำหรับ internal และสีดำสำหรับ external และปุ่มที่ให้กดสำหรับ upload skill จะอยู่ในหน้านี้ด้วยโดยมีการอัพทั้งหมด 2 แบบคือ internal และ external โดยที่ internal คือการให้อัพ skill.md โดยการอัพโหลด file โดยตรง แต่ external คือการให้คนกดแนบลิ้ง github และมันทำการ traversal ทุก folder ภายในว่ามี skill ไหมโดยถ้าเจอ 3 สกิลก็ให้อัพแยกขึ้นเป็น 3 สกิลและชื่อเจ้าของ skill ให้ใช้เป็นชื่อของผู้อัพ และในหน้า marketplace และไม่ว่าจะเป็น internal หรือ external ข้อมูลที่เราเก็บจะเป็น folder ทั้งหมดและทางเราจะเก็บทั้ง folder เพื่อให้ตอนกดโหลดได้ file จริง และเมื่อกดที่สกิลไหนก็จะสามารถเข้าไปที่หน้า skill ของ skill นั้นๆได้ ส่วนสกิลทุกสกิลจะมีรูปดาวแปะอยู่ขวาบนหากกดจะถือว่าเราชอบสิ่งนั้นและเพิ่มนั้นหนักให้การแนะนำสกิลภายหลังและเราสามารถ filter เพื่อดูแต่สกิลที่เรากดดาวได้(ดาวกดซ้ำเพื่อเลิกได้เป็นปุ่ม toggle) ลืมบอกสกิลที่แนะนำ กับสกิลที่ลิสทั้งหมดนี่คือแบ่งเป็นคนละโซนนะ ในหน้านี้ให้มันแสดงทั้งคู่แหละ
3. หน้า skill จะเห็น file ทั้งหมดของ folder ของสกิลนั้นๆว่ามี file อะไรบ้างและสามารถกดเข้าไปดูเนื้อหาของ file นั้นๆได้และสามารถกด download ลงมาได้แต่การกด download จะให้ทั้ง folder เลยไม่สามารถเลือกเฉพาะ file ได้ และหากคนที่กดเข้ามาดูเป็นคนที่โพสสกิลนั้นหรือเป็น admin จะสามารถแก้ไขเนื้อหาของ skill เพิ่มลด tag หรือ category ได้
4. timeline เป็นหน้าแสดง chart โดยจะมีทั้งหมด 5 chart ดังนี้ 1. skill upload overtime แสดงเป็นกราฟเส้น(กดให้กลายเป็นกราฟแท่งได้)โดยบอกปริมาณสกิลที่ถูกอัพวันนั้น และแบบสะสมตลอด lifetime 2. recent upload แสดง 5 สกิลที่พึ่งถูกอัพโหลดตรงๆ 3. Skill distribution by category กราฟวงกลมพร้อมบอกอัตราส่วน skill ว่ามี tag ใดบ้างพร้อมบอกเปอเซ้น 4. top tag บอกชื่อ tag ที่มีในระบบมากที่สุดพร้อมจำนวนสกิลมา 10 tag 5. กราฟวงกลมเหมือน Skill distribution by category แต่รอบนี้บอกเป็น อัตราส่วน internal กับ external แทน
5. tag management 1.โดยทุกคนสามารถ add  tag category ให้สกิลได้ที่เป็นเจ้าของได้(ถ้าไม่ใช่เจ้าของจะมองไม่เห็น แต่ถ้าเป็น admin จะทำได้ไม่สนว่าเป็นเจ้าของไหม) 2.ถ้าเป็น admin สามารถสร้างหรือลบ tag category และ group ของระบบได้เลย(user ปกติมองไม่เห็น)
6. permission 1. admin สามารถเพิ่มสมาชิกเข้ากลุ่มต่างๆได้ 2.ทุกคนสามารถปรับสกิลเป็น public และ private ได้ ถ้าเป็น private สามารถเพิ่ม group ได้เลย(เฉพาะเจ้าของ skill นั้นๆหรือ admin เท่านั้น)
